Introducción

  • Moodle : SASISOPA
  • Version de documentación: v 1.0

Es muy importante conocer la documentación para poder hacer uso de las mejoras a la plataforma de SASISOPA.
Por favor asegurese de seguir los pasos para el correcto funcionamiento de la plataforma.

Requirementos

Necesitarás configurar lo siguiente para la correcta extraccion de los PDF.

  1. Configurar correctamente la url de tu dominio.
  2. Configurar la plantilla html principal con las clases y id correctos.
  3. Crear una carpeta para guardar las firmas.
  4. Crear una carpeta para guardar los PDF generados.
  5. Configuar la base de datos deseada para habilitar la extracción.

Es muy impoortante seguir las instrucciones de como nombrar las carpetas,
ya que si se nombran de diferente manera, no funcionará correctamente la extracción.

Configurando la URL

Ingresa a tu servidor, a la ruta donde se encuentren las carpetas de moodle. Una vez dentro de la carpeta principal de moodle, buscar la siguiente ruta:

https://tudominio.com/sasisopa/mod/data/url.php

Abrir el archivo url.php y modificar la url existente colocando la tuya, en éste ejemplo moodle se encuentra dentro de la carpeta SASISOPA por lo que se coloca dentro de la url 

                       
                          $urlImagen = "https://tudominio.com"

Aquí un ejemplo. en caso de moodle se encuentre dentro de otra carpeta, es necesario colocar el nombre de esta carpeta como se muestra en la siguiente imagen 

                                    $urlImagen = "https://tudominio.com/tucarpeta"

Firmas

Configuración de firmas

Para poder tener disponibles las firmas para los documentos a exportar en PDF, es necesario crear una carpeta en la sección principal con "modo activar edición" habilitado.

Pasos a seguir:

1.- Añadir una actividad o recurso y selecciona una carpeta.

2.- Nombra la carpeta “Firmas” sin comillas.

PDF

Configuración para extracción PDF

Requirementos

Necesitarás configurar lo siguiente para la correcta extraccion de los PDF.

  1. Es necesario agregar una carpeta nueva en donde se guardarán los PDF extraídos, por lo que esta carpeta debe generarse dentro de la sección deseada.
  2. Configurar la base de datos para habilitar extracción de PDF agregando un campo nuevo

Pasos a seguir:

1.- Añadir una actividad o recurso y selecciona una carpeta.

2.- Nombra la carpeta

Ingresar en el campo de nombre lo siguiente: “Evidencias PDF + nombre de la base de datos a extraer” sin comillas.
Ejemplo:
Evidencias PDF Cumplimiento de Requisitos Legales

una vez conluidos los pasos anteriores, es necesario agregar un campo nuevo para habilitar a la base de datos deseada para la extracción.

  • * seleccionar "crear un nuevo campo" y luego “casilla de seleccion”.
